## VALUE SCALING REFERENCE

### BPM Scaling (Channel 2, CC 50-53)
- 0 = 60 BPM
- 64 = 120 BPM (typical range center)
- 127 = 180 BPM
- Formula: MIDI_Value = ((BPM - 60) / 120) * 127

### Key Detection Scaling (Channel 2, CC 70-73)
- 0 = A (1A)
- 11 = A♭/G♯ (12A)
- 12 = A (1B)
- 23 = A♭/G♯ (12B)
- Camelot notation mapping included

### EQ Center Values (Channel 1, CC 12-27)
- 0 = Full cut (-26dB)
- 64 = Center/flat (0dB)
- 127 = Full boost (+12dB)

### Filter Center Values (Channel 1, CC 24-27)
- 0 = Full low-pass
- 64 = No filter
- 127 = Full high-pass

### Tempo Range (Channel 1, CC 42-45)
- 0 = -100% (0.5x speed)
- 64 = 0% (normal speed)
- 127 = +100% (2x speed)

## MODIFIER STATES
- M1: Shift function (Hold = 1, Release = 0)
- M2: AI Control Enable (Off = 0, On = 1)
- M3: Deck A Override (AI = 0, Human = 1)
- M4: Deck B Override (AI = 0, Human = 1)
- M5: Deck C Override (AI = 0, Human = 1)
- M6: Deck D Override (AI = 0, Human = 1)

## SAFETY PROTOCOLS
1. Emergency Stop (CC 80): Immediate stop, no fade
2. Volume Limiting (CC 86): Hard limit on master volume
3. Manual Override: Individual deck control priority
4. AI Enable (CC 81): Master switch for AI control
5. Real-time Monitoring: Continuous status feedback

## COMMUNICATION PROTOCOL
- Input Latency Target: <10ms
- Output Update Rate: 40Hz (25ms intervals)
- MIDI Channel Priority: 3 > 1 > 2 > 4
- Buffer Size: 128 samples recommended
- Thread Priority: Real-time for channels 1 & 3

## RESERVED CC RANGES
- 0-6: System reserved
- 118-126: Future expansion
- 127: Modifier controls only
